# Nikskart

A modern React + Vite e-commerce website.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development:
   ```bash
   npm run dev
   ```

3. Run the backend server in a second terminal:
   ```bash
   npm run server
   ```

4. Build for production:
   ```bash
   npm run build
   ```
